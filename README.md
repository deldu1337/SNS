# 익스프레스를 활용한 SNS 웹 서비스
# Facebook, Instagram, Twitter와 같은 Sns에 존재하는 기본적인 기능이 포함된 간단한 SNS 웹 서비스
#### 프로젝트 개발 기간 : 2022.04 ~ 2022.05
#### 프로젝트 개발 인원 : 4인
-----------------------
## Description
### PART 1. 프로그램 제공 기능
 - 게시물 등록/수정/삭제 기능
 - 좋아요 기능
 - 팔로우 및 팔로우 취소 기능
 - 프로필 사진 등록 및 변경 기능
 - 상태 메시지 등록 및 수정 기능
 - 내 게시물 보기 기능
 - 비밀번호 찾기 기능
 - 테마 색상 변경 기능
